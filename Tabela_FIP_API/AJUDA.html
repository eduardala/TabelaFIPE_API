Seja bem-vindo ao AJUDA do trabalho referente à tabela FIPE.

O projeto é formado pelos seguintes packages:

DTO
      O pagckage DTO é composta por 4 classes, Marca, Modelo, AnoModelo e VeiculoCompleto, as classes armazenadas neste package são apenas para dar início a utilização das listas.
 

FIPE
      O package FIPE possui uma classe chamada Fipe, esta classe torna possível a chamada dos métodos necessários referente a criação das listas a serem utilizadas para chegar a um devido resultado. Os métodos receberão como parâmetro as suas devidas informações para obter o resultado, todos os métodos possuem o parâmetro "tipoLista" que deverá receber como padrão marca, modelo, anomodelo ou veiculo, quando seu devido método for chamado. Os métodos recebem os parâmetros, como por exemplo idMarca, idVeiculo, idModelo.
 

Fabrica
      O package Fabrica, possui as seguintes classes, FactoryMarca, FactoryModelo, FactoryAnoModelo e FactoryVeiculoCompleto, que irá realizar a criação das fábricas conforme forem chamadas, todas as classes possuem um "implements" da classe IFactory.       
 

InterfaceFabricas
      O package InterfaceFabricas possui uma interface chamada IFactory, está interface possui apenas um método para realizar a criação das listas, recebendo como parâmetro uma URL.
 

Persistencia
      O package Persistencia possui uma clase chamada ManipulandoJSON, nesta classe será possível realizar a exportação das listas e limpa-las realizando a chamadas dos métodos que essa classe possui.
 

WebService
      O package WebService possui uma classe chamada Conexao, essa classe possui um método chamado verifica(), este método irá verificar a conexão para consumir o WebService, se o método retornar true, será possível consumir o WebService, se o método retornar false, não será possível.
